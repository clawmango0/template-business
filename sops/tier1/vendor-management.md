# SOP: Vendor Management

**Company:** {{COMPANY_NAME}}
**Department:** Procurement / Operations
**Document ID:** VM-001

---

## 1. Purpose

Establish a consistent process for selecting, evaluating, and managing vendors to ensure quality, compliance, and cost-effectiveness.

## 2. Scope

Applies to all vendors providing:
- Goods (equipment, supplies)
- Services (consulting, maintenance)
- Software/SaaS subscriptions

## 3. Vendor Categories

| Category | Examples | Review Frequency |
|----------|----------|-----------------|
| Critical | Core software, infrastructure | Quarterly |
| Standard | Office supplies,常规 services | Annually |
| Low | Miscellaneous | Bi-annually |

---

## 4. Vendor Selection Process

### 4.1 Identification

1. Identify need
2. Document requirements
3. Research potential vendors (3 minimum)

### 4.2 Evaluation Criteria

| Criterion | Weight |
|-----------|--------|
| Cost | % |
| Quality | % |
| Reliability | % |
| Support | % |
| Compliance | % |
| **Total** | 100% |

### 4.3 Evaluation Matrix

| Vendor | Cost Score | Quality Score | ... | Total Score |
|--------|------------|---------------|-----|-------------|
| Vendor A | | | | |
| Vendor B | | | | |
| Vendor C | | | | |

### 4.4 Selection

- Score and rank vendors
- Negotiate terms
- Obtain approvals per approval matrix

## 5. Vendor Onboarding

- [ ] Complete W-9
- [ ] Sign vendor agreement
- [ ] Set up in accounting system
- [ ] Add to approved vendor list
- [ ] Conduct security review (if applicable)

---

## 6. Ongoing Management

### 6.1 Performance Reviews

| Metric | Target | Measurement |
|--------|--------|-------------|
| Quality | % | Defect rate |
| Delivery | % | On-time % |
| Response | X hrs | Avg response time |
| Cost | $ | Budget vs actual |

### 6.2 Issue Resolution

1. Document issue
2. Contact vendor
3. Escalate if unresolved
4. Consider replacement

---

## 7. Offboarding

- [ ] Cancel subscriptions
- [ ] Retrieve company assets
- [ ] Remove from systems
- [ ] Archive records
