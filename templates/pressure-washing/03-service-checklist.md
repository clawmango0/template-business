## SERVICE CHECKLIST

### Surface Cleaning Order

#### 1. Pre-Treatment
- [ ] Apply degreaser to oil stains
- [ ] Apply mold/mildew treatment
- [ ] Wait [5-15] minutes
- [ ] Agitate stubborn areas

#### 2. Surface Cleaning
- [ ] Start from top, work down
- [ ] Use appropriate pressure
- [ ] Overlap strokes [30]%
- [ ] Keep nozzle consistent distance
- [ ] Clean in sections

#### 3. Driveway
- [ ] Apply degreaser
- [ ] Use surface cleaner
- [ ] Target oil stains
- [ ] Clean edges and corners
- [ ] Rinse thoroughly

#### 4. Walkways/Patios
- [ ] Apply cleaner
- [ ] Surface clean
- [ ] Detail edges
- [ ] Clean transitions

#### 5. Siding
- [ ] Start at bottom (gravity)
- [ ] Work in sections
- [ ] Maintain wet edge
- [ ] Clean windows last

#### 6. Fence
- [ ] Apply cleaner
- [ ] Use appropriate pressure
- [ ] Work with wood grain
- [ ] Rinse before drying
