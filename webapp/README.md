# Template Manager Web Application

A simple web app to create, store, and retrieve filled templates.

## Setup

```bash
# Install dependencies
pip install flask

# Run the app
python app.py
```

## Features

- Create new forms from templates
- Fill in template fields
- Save forms to database
- Search and retrieve previous forms
- Export to Markdown/Word

## Access

Open http://localhost:5000 in your browser.
