# User Guide: Executive & Strategy Templates

## Overview

Executive templates help leaders communicate effectively with boards, investors, and internal teams. Use these for strategic planning and high-level communication.

| Use | Audience |

---

## How to Use Each Template

### 1. Board Update

**Purpose:** Keep board/investors informed on company progress.

**Frequency:** Monthly or quarterly

**Audience:** Board members, investors, executives

**Key sections:**
- Executive summary (2-3 sentences)
- Highlights (wins)
- Challenges (issues)
- Key metrics
- Financial overview
- Product updates
- Team updates
- Ask/decisions needed

**Best practices:**
- Keep it concise (5-7 slides/pages)
- Lead with the most important
- Be transparent about challenges
- Include clear asks
- Send 2-3 days before meeting

### 2. QBR (Quarterly Business Review)

**Purpose:** Comprehensive quarterly review of business performance.

**Frequency:** Quarterly

**Audience:** Executive team, department heads

**Sections:**
- Financial performance
- Customer success
- Product/technology
- Team/culture
- Goals for next quarter

**Process:**
1. Collect data from all departments
2. Compile into template
3. Present in leadership meeting
4. Identify action items
5. Document follow-ups

### 3. OKR Tracker

**Purpose:** Track company, team, and individual objectives.

**Frequency:** Quarterly with weekly check-ins

## Best Practices

### Board Updates
- [ ] Send in advance (2-3 days)
- [ ] Practice presentation
- [ ] Anticipate questions
- [ ] Follow up on asks
- [ ] Archive previous updates

### QBRs
- [ ] Prepare 1 week ahead
- [ ] All depts present
- [ ] Focus on decisions needed
- [ ] Document action items
- [ ] Distribute summary after

### OKRs
- [ ] Set at start of quarter
- [ ] Review weekly
- [ ] Grade honestly
- [ ] Carry forward incomplete
- [ ] Celebrate achievements

## Common Mistakes

| Template | Mistake | Solution |
|----------|---------|----------|
| Board Update | Too detailed | Keep high-level |
| Board Update | Burying bad news | Lead with challenges |
| QBR | No follow-through | Document actions |
| QBR | Missing data | Prepare ahead |
| OKRs | Too many | Limit to 3-5 |
| OKRs | Vague KRs | Make measurable |

## Presentation Tips

### Board Presentations
1. Lead with metrics
2. Show trajectory
3. Be honest about challenges
4. Make asks clear
5. Leave time for questions

### QBR Presentations
1. Review previous quarter
2. Celebrate wins
3. Address misses
4. Set next quarter goals
5. Assign owners
