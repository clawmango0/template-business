# User Guide: Finance & Admin Templates

## Overview

Finance templates help manage money, track assets, and maintain fiscal responsibility. Use these tools to stay organized and professional.

---

## Template Overview

| Template | When to Use | Purpose |
|----------|-------------|---------|
| Invoice | Billing clients | Get paid |
| Expense Report | Employee reimbursements | Track spending |
| Budget Tracker | Planning & monitoring | Stay on budget |
| Equipment Inventory | Asset tracking | Prevent loss |

---

## How to Use Each Template

### 1. Invoice

**Purpose:** Request payment for goods/services.

**When to create:**
- Completed project
- Delivered service
- Monthly billing cycle

**Key fields:**
- Invoice number (sequential)
- Date and due date
- Client information
- Line items with pricing
- Total and payment instructions

**Best practices:**
- Number invoices sequentially
- Specify payment terms (Net 30)
- Include payment methods
- Send promptly upon delivery

### 2. Expense Report

**Purpose:** Reimburse employees for business expenses.

**When to use:**
- Employee travels for work
- Business-related purchases
- Client entertainment

**Required documentation:**
- Itemized receipts
- Business purpose
- Approval from manager

**Process:**
1. Employee fills out form
2. Attaches receipts
3. Manager approves
4. Finance reimburses
5. Records maintained

### 3. Budget Tracker

**Purpose:** Monitor actual vs planned spending.

**When to use:**
- Annual planning
- Department budgets
- Project budgets

**How to use:**
1. Set budget at start of period
2. Track actuals monthly
3. Analyze variances
4. Adjust forecasts

**Variance analysis:**
| Variance | Action |
|----------|--------|
| >10% over | Review and adjust |
| 5-10% over | Monitor |
| On track | Continue |

### 4. Equipment Inventory

**Purpose:** Track company assets.

**When to use:**
- New equipment purchases
- Employee device assignments
- Audit preparation
- Exit interviews

**Tracking categories:**
- Computing (laptops, monitors)
- Mobile devices
- Furniture
- Infrastructure

## Best Practices

### Invoicing
- [ ] Use professional template
- [ ] Clear line items
- [ ] Prompt submission
- [ ] Follow up on overdue

### Expenses
- [ ] Pre-approve large purchases
- [ ] Receipt required >$25
- [ ] Timely submission (30 days)
- [ ] Clear business purpose

### Budgets
- [ ] Realistic targets
- [ ] Monthly review
- [ ] Document variances
- [ ] Adjust as needed

### Inventory
- [ ] Unique asset tags
- [ ] Regular audits (quarterly)
- [ ] Clear assignment records
- [ ] Proper disposal process

---

## Common Mistakes

| Mistake | Impact | Solution |
|---------|--------|----------|
| No invoice numbering | Tracking issues | Sequential numbering |
| Missing receipts | Audit problems | Require always |
| Unrealistic budgets | Missed targets | Data-driven |
| No asset tags | Lost equipment | Tag everything |
