## üìã SOPs (Standard Operating Procedures)

### When the template says: "Write a step-by-step procedure"

**AI Prompt to use:**
```
You are an operations expert. Create a detailed step-by-step SOP for [TOPIC]. 
Include:
- Objective (why this matters)
- Prerequisites (what's needed before starting)
- Step-by-step instructions (numbered, clear)
- Quality checkpoints (how to know it's done right)
- Troubleshooting tips (common issues and solutions)

Context: My company is [INDUSTRY], team size is [SIZE]. 
Make it practical for [EXPERIENCE LEVEL] employees.
```

### When the template says: "Document the process"

**AI Prompt:**
```
Create a comprehensive process document for [PROCESS NAME].
Include:
- Flowchart description (decision points)
- Role responsibilities (who does what)
- Time estimates for each step
- Required tools/resources
- Success criteria

Make it clear enough for a new employee to follow without additional guidance.
```

## üë• HR Templates

### When the template says: "Write job description"

**AI Prompt:**
```
Create a professional job description for [JOB TITLE] at [COMPANY TYPE].
Include:
- Position overview (2-3 sentences)
- Key responsibilities (5-7 bullet points)
- Required qualifications (hard skills)
- Preferred qualifications (nice to have)
- Compensation range (market rate for [LOCATION])
- Benefits highlights
- How to apply

Make it compelling to attract [TARGET CANDIDATE TYPE] while being accurate.
```

### When the template says: "Document company policies"

**AI Prompt:**
```
Write a company policy for [POLICY TOPIC] that is:
- Compliant with employment law in [STATE]
- Clear and easy to understand (avoid legal jargon)
- Practical for our [INDUSTRY] company with [EMPLOYEE COUNT] employees

Include:
- Purpose statement
- Who this applies to
- The policy (clear rules)
- Procedures for implementation
- Consequences of non-compliance
- Review frequency

Add a brief section explaining WHY this policy matters to employees.
```

## üìä Project Management

### When the template says: "Define project scope"

**AI Prompt:**
```
Create a comprehensive project charter for [PROJECT NAME].
Include:
- Project vision (one sentence)
- Business case (why this matters now)
- Scope: What's IN and OUT of bounds
- Key stakeholders and their interests
- Major milestones with target dates
- Resource requirements
- Risks and mitigation strategies
- Success metrics
- Approval/sign-off section

This is for a [INDUSTRY] project with [TEAM SIZE] people.
```

### When the template says: "Create RACI matrix"

**AI Prompt:**
```
Create a RACI matrix for [PROJECT OR PROCESS].
Include these roles: [LIST ROLES].
For each major task/decision, assign:
- R = Responsible (does the work)
- A = Accountable (final decision)
- C = Consulted (provides input)
- I = Informed (kept updated)

Format as a clear table. Flag any potential conflicts (someone with multiple R's).
```

## üéØ Executive Templates

### When the template says: "Write executive brief"

**AI Prompt:**
```
You are an executive communications expert. Help me write a compelling executive brief.

Context:
- Purpose: [BOARD UPDATE / INVESTOR UPDATE / STRATEGIC PLAN]
- Audience: [BOARD / INVESTORS / EXECUTIVE TEAM]
- Key message: [MAIN THESIS]
- Supporting points: [2-3 KEY SUPPORTS]
- Desired outcome: [WHAT DO YOU WANT FROM AUDIENCE]

Create:
1. Executive summary (2-3 sentences that capture attention)
2. Key points (with data/support)
3. Challenges (honest assessment)
4. Ask (what you need from them)

Make it concise, data-driven, and action-oriented.
```

### When the template says: "Define OKRs"

**AI Prompt:**
```
Help me set effective OKRs for [QUARTER/YEAR] at [COMPANY TYPE].

Current priorities: [LIST 3-5 PRIORITIES]

For each objective, create 3 measurable KRs using the format:
- Objective: [What we want to achieve]
  KR1: [Metric] = [Target]
  KR2: [Metric] = [Target]  
  KR3: [Metric] = [Target]

Make KRs:
- Ambitious but achievable (70% = success)
- Quantifiable (can measure progress)
- Time-bound (can achieve in [TIMEFRAME])

Add a brief guide on how to track and grade these quarterly.
```

## üîß How to Use These Prompts

### Step-by-Step Process:

1. **Copy** the prompt template above
2. **Replace** the bracketed placeholders with your specifics
3. **Paste** into your preferred AI tool:
   - Grok (x.com/Grok)
   - ChatGPT (chat.openai.com)
   - Claude (claude.ai)
   - Copilot (copilot.microsoft.com)
4. **Review** the AI output
5. **Edit** to match your voice and needs
6. **Insert** into your template

### Tips for Best Results:

| Tip | Why |
|-----|-----|
| Be specific | More context = better output |
| Iterate | First draft ‚Üí refine ‚Üí final |
| Add constraints | "In under 500 words" or "Use simple language" |
| Ask for options | "Give me 3 versions" |
| Request format | "Output as bullet points" or "As a table" |
